## 1. Project Overview

This project is a fully responsive multi-page website created for **QuickFix Plumbing**, a fictional plumbing service provider based in Melbourne. The website demonstrates competency in HTML, CSS, responsive design, image optimisation, interactivity, and client-side form validation as required in **Part 2** and **Part 3** of the Web Development (Introduction) module.

The website includes the following pages:
- `index.html` (Home)
- `about.html` (About Us)
- `services.html` (Services)
- `contact.html` (Contact/Enquiry)
- `css/styles.css`, `js/script.js`

## 2. Part 2 – Requirements Completed

### External Stylesheet
All pages link to `/css/styles.css`.

### CSS Selectors & Base Styles
- Universal reset using `*`
- Organized CSS structure with semantic section comments
- Color palette and typography applied consistently

### Typography
- Scalable text sizes
- Semantic heading hierarchy (`h1–h3`)

### Desktop Layout (Flexbox / Grid)
- Flexbox used for header & navigation
- CSS Grid used for service / gallery / team layouts

### Visual Styling
- Navigation hover effects
- Buttons and interactive elements styled
- Professional spacing and layout

### Responsive Web Design
Media queries implemented for:
- Tablet (max-width: 900px)
- Mobile (max-width: 480px)

These adjust:
- Navigation stacking  
- Section spacing  
- Layout reflow  
- Font scaling  

### Responsive Images
- All images now scale proportionally with `.responsive-img` class
- `picture` element used for logos and hero images where appropriate
- Alt text included for accessibility

### Testing
Tested on:
- Desktop (1920px)
- Tablet (768px)
- Mobile (375px)
